# LIVE_TRANSITION_BLOCKERS

안전한 live 전환을 아직 막는 요인을 고정합니다. (2026-03-28 기준)

## Critical Blockers
1. 브로커별 실주문 체결 편차 검증 미완료
- 모의 체결 모델과 실브로커 체결 로그 간 차이(체결가/부분체결/체결지연) 계량화가 필요.

2. 이벤트 급변 구간 슬리피지 모델 실측 교정 미완료
- FOMC/CPI/고용 등 이벤트 윈도우에서 현재 모델의 오차가 큰 구간 존재 가능.

3. Shadow mode 장기 관찰 데이터 부족
- 실주문 미집행 shadow 모드로 최소 수 주 이상 시그널/체결 시뮬 결과 축적 필요.

4. 운영 통제(알람/kill-switch/감사로그) 검증 미완료
- 비정상 주문/연속 손실/브로커 응답 장애 대응 절차와 자동 차단 체계가 실운영 수준으로 검증되지 않음.

## Recommended Exit Criteria
- 브로커별 shadow/live-sim 추적 오차가 사전 정의 임계치 이내.
- 이벤트 데이 샘플 포함 슬리피지 모델 재교정 완료.
- kill-switch 및 운영 알람 시나리오 리허설 통과.
- 감사로그(누가/언제/왜/무엇을 주문했는지) 저장 및 조회 흐름 확인.
